== Description ==

Sections is a WordPress plugin that gives you an alternative to the default Widgets. Sections are pieces of content to be used as shortcode or directly in your theme.

== Installation ==

1. Upload the folder `sections` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
