== Description ==

Sections is a WordPress plugin that gives you an alternative to the default Widgets. Sections are pieces of content to be used as shortcode or directly in your theme.

== Installation ==

1. Upload the folder `sections` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> Sections to configure the custom post type to be used as section.

The custom post type must be created by the user in the functions.php, or a plugin that can create custom post types withoud programming.
